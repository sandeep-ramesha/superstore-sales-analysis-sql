# Superstore Sales Analysis using SQL

## 📌 Overview
This project analyzes sales, profit, and customer trends using SQL on the Superstore dataset.

## 🛠️ Tools Used
- SQL
- Data Analysis

## 📊 Key Analysis Performed
- Total sales and profit analysis
- Regional and state-wise performance
- Identification of loss-making states
- Sub-category and segment analysis
- Discount impact analysis

## 📈 Key Insights
- Identified regions and states with highest and lowest profitability
- Detected loss-making products and sub-categories
- Analyzed customer segments contributing to revenue

## 📁 Files
- SQL Project Presentation (PPT)
- SQL Queries and outputs

## 🚀 Outcome
This project demonstrates strong SQL skills in querying, aggregation, filtering, and business analysis.
